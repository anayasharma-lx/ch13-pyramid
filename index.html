Each bottom value reaches the top with a fixed weight — the binomial coefficient from Pascal's triangle. The corner values have weight 1 (one path to the top). Middle values have larger weights (more paths). Changing any bottom value changes the top by exactly its weight.

Pascal's triangle — the highlighted row gives the pyramid weights

For a pyramid with n cells at the bottom, the weights are row n-1 of Pascal's triangle. The sum of weights = 2^(n-1). Put consecutive Fibonacci numbers in the base — the top is always another Fibonacci number!
